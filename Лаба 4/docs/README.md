# Упрощенная банковская система

Этот проект реализует консольное приложение **"Упрощенная банковская система"**, разработанное в рамках лабораторной работы №4 по дисциплине "Обработка данных и искусственный интеллект".

## Структура проекта

```
BankSystem/
├── include/          # Заголовочные файлы (.h)
├── src/             # Файлы реализации (.cpp)
├── docs/            # Документация
├── build/           # Папка для объектных файлов (создается автоматически)
├── Makefile         # Файл сборки с использованием make
└── CMakeLists.txt   # Файл конфигурации для CMake
```

## Компиляция проекта

Существует два способа собрать проект: используя **Makefile** или **CMake**. Для обоих вариантов требуется установленный компилятор C++ с поддержкой стандарта C++17 (например, `g++`).

### Использование Makefile

Выполните следующие команды из корневой папки проекта `BankSystem`:

```sh
make          # Собирает проект в исполняемый файл `bank_system`
./bank_system # Запускает собранное приложение
make clean    # Очищает объектные файлы и исполняемый файл
```

### Использование CMake

```sh
mkdir -p build
cd build
cmake ..       # Генерирует файлы сборки
cmake --build . # Собирает исполняемый файл `bank_system`
./bank_system   # Запускает приложение
```

## Запуск программы

После запуска исполняемого файла `bank_system` в консоли появится меню, позволяющее:

- Добавлять, редактировать и удалять клиентов (обычных и премиум);
- Открывать, закрывать и просматривать счета;
- Выполнять операции пополнения, снятия и перевода средств;
- Просматривать историю транзакций и статистику банка.

Следуйте подсказкам в меню для ввода необходимой информации. Для завершения работы выберите пункт меню "0. Exit".

## Зависимости

Проект не использует сторонних библиотек и написан на стандартном C++17. Для корректной работы необходим только компилятор C++ и инструменты сборки (make или CMake).
